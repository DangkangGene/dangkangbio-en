## 1. Introduction

### 1.1 Purpose of This Checklist

This checklist supports Australian medical testing laboratories in preparing for NATA accreditation and provides guidance for manufacturers (like Dangkang Bio) whose products may be used in NATA-accredited laboratories. It covers requirements under:

- **ISO 15189:2022** - Medical laboratories - Requirements for quality and competence
- **NPAAC Standards** - National Pathology Accreditation Advisory Council requirements
- **TGA Requirements** - For in-house IVD use

### 1.2 How to Use This Checklist

- Review each requirement systematically
- Document current compliance status
- Identify gaps requiring corrective action
- Assign responsibility and target dates
- Retain evidence of compliance

## 5. Quality Control Requirements

### 5.1 Internal Quality Control (IQC)

| QC Component | Requirement | Frequency | Status | Notes |
|--------------|-------------|-----------|--------|-------|
| Negative control | Every run | Per run | ☐ | |
| Positive control | Every run | Per run | ☐ | |
| Internal control | If applicable | Per run | ☐ | |
| Levey-Jennings chart | QC data plotted | After each QC | ☐ | |
| Westgard rules | Applied | Per QC event | ☐ | |
| QC rules violation | Action documented | Per incident | ☐ | |

**Westgard Rules to Apply:**
- 1:2s - Warning rule (1 control > 2 SD)
- 1:3s - Reject (1 control > 3 SD)
- 2:2s - Reject (2 consecutive > 2 SD same side)
- R:4s - Reject (range > 4 SD)
- 4:1s - Reject (4 consecutive > 1 SD same side)
- 10x - Reject (10 consecutive same side)

## 10. Quick Reference Summary

### 10.1 Most Common NATA Findings

Be particularly attentive to these commonly cited deficiencies:

1. **Documentation issues**
   - Outdated SOPs
   - Missing approval signatures
   - Incomplete records

2. **QC deficiencies**
   - QC rules not applied correctly
   - QC trending not reviewed
   - Outliers not investigated

3. **Staff competency**
   - Incomplete competency assessments
   - Training records incomplete
   - Authorization not documented

4. **Equipment issues**
   - Calibration overdue
   - Maintenance not documented
   - IQ/OQ/PQ incomplete

5. **Validation/verification**
   - Validation incomplete
   - Acceptance criteria not met
   - Report deficiencies

### 10.2 Success Factors

**Key to Successful Accreditation:**
- Executive commitment and resources
- Dedicated quality management focus
- Staff engagement and training
- Realistic timeline
- Documentation discipline
- Continuous improvement culture
